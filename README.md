# Gujarat Pharmaceutical Industry Analytics

**Enterprise-Scale SQL Server Database Project**

---

## Overview

A fully normalized SQL Server database modeling pharmaceutical industry operations across Gujarat, India.

Built to demonstrate advanced SQL engineering, performance optimization, and business-driven analytics within a realistic enterprise data environment.

---

## Scope

* 100+ pharmaceutical companies
* 15 relational tables
* 100+ analytical queries (Beginner → Expert)
* Industry-aligned business scenarios

---

## Data Domains Modeled

* Company & facility management
* Product portfolio
* Sales & export operations
* Research & clinical trials
* Regulatory compliance
* Quality assurance
* Supplier & procurement analytics
* Workforce data
* Financial performance
* Market intelligence

---

## Technical Stack

* Microsoft SQL Server 2019+
* SQL Server Management Studio (SSMS)

---

## Advanced SQL Implementation

* Multi-table JOIN strategies
* Correlated and nested subqueries
* Common Table Expressions (CTEs)
* Window functions (ROW_NUMBER, RANK, LAG, LEAD)
* KPI-driven aggregations
* Trend and ranking analytics

---

## Performance Engineering

* Clustered and non-clustered indexing strategies
* Query execution plan analysis
* Index optimization for JOIN and filtering columns
* Reduced logical reads and improved query response time
* Structured query refactoring for scalability

This demonstrates practical understanding of production-level SQL performance tuning — not just query writing.
